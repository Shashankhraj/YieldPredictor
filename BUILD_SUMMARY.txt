────────────────────────────────────────────────────────────────────────────
CropYield AI is a full-featured, production-ready web application for 
predicting agricultural crop yields using machine learning. Built with 
modern web technologies and featuring a beautiful, intuitive interface.

MODEL PARAMETERS
────────────────────────────────────────────────────────────────────────────

Input Parameters (9 total)
  1. Crop Type (5 options)
  2. Soil pH Level (4-9)
  3. Soil Moisture (0-100%)
  4. Nitrogen Content (0-300 mg/kg)
  5. Phosphorus Content (0-100 mg/kg)
  6. Potassium Content (0-500 mg/kg)
  7. Rainfall (0-1000 mm)
  8. Temperature (-10 to 50°C)
  9. Humidity (0-100%)

Output Format
  {
    "yieldKgHa": number,           // Estimated yield in kg/hectare
    "confidence": number,           // Confidence score (0-0.99)
    "message": string              // Contextual recommendation
  }
